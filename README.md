# Silent Waste Detector

AI-based project to detect waste-prone areas from images using computer vision and deep learning.

This system analyzes images and classifies them into clean, moderate, or high waste zones based on object detection and scene assessment logic.

## Features
- Supports image formats including JPG, PNG, and JFIF.
- Uses YOLOv8 object detection and deep learning.
- Lightweight and easy to run with Python.
